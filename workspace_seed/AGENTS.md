## Preferences
- 사용자는 주식 모니터링 및 메일 알림 자동화를 선호한다.
- LangChain 기반 에이전트가 스킬을 통해 작업을 수행하도록 구성한다.
- 알림은 기준치(전일 종가, 20일선)와 비교해 5% 이상 차이날 때만 보내도록 한다.

## Notes
- TIGER 미국나스닥 100 종목 코드: 133690
- 모니터링 주기: 매일 영업일 오전 10시, 오후 3시
- 기준 비교값: 전일 종가, 20일선
- 임계값: 5% 이상
- 알림은 텔레그램 봇 토큰과 채팅 ID 설정을 사용한다.
- 실제 주가 데이터 수집은 향후 API 연동으로 교체할 수 있다.
- 모니터링 작업은 workspace/skills/tiger-nasdaq-monitor/manage_tiger_monitor.py 를 사용한다.
